Cannon Craze Privacy Policy

Effective date: July 8, 2026

Cannon Craze ("the game") is developed and published by Anas Uddin ("the developer"). This policy describes what information the game handles. The short version: the game collects nothing.

Data collection

Cannon Craze does not collect, store, transmit, sell, or share any personal data or any other data. Specifically, the game:

  • requests no permissions on any platform
  • makes no network connections of its own
  • contains no advertising, no analytics, and no third-party SDKs beyond the open-source Processing rendering runtime, which also makes no network connections
  • has no accounts, no sign-in, and no user-generated content

Local storage

The game saves your best score and your settings (ball size, target count, trajectory guide, sound, volume, graphics quality) on your device only:

  • Desktop: two small text files next to the game (data/high_score.txt, data/settings.txt)
  • Android: the app's private preferences storage

This data never leaves your device. Uninstalling the game (or deleting the files) removes it completely.

This website

This website is a static page. It sets no cookies, runs no analytics, and embeds no third-party trackers. It is hosted on Vercel, whose infrastructure may keep standard, short-lived server logs (such as IP addresses) to operate the service, as described in Vercel's privacy policy. Download links point to GitHub Releases and are subject to GitHub's privacy statement.

External links

The game's credits screen contains a button that opens the developer's GitHub page in your browser. Following that link is subject to GitHub's own privacy policy.

Children

The game contains no chat, no purchases, no ads, and collects no data from anyone, including children.

Changes to this policy

If a future version of the game ever changes any of the above, this policy will be updated and the effective date revised before that version ships.
