# Cannon Craze

A minimal, neon-noir arcade cannon game built with [Processing](https://processing.org) (Java).

Pull the glowing cannonball back like a slingshot, release to fire, and land on the lit pad.
One miss ends the run — beat your best score.

## How to play

1. **Grab** the glowing cannonball at the muzzle.
2. **Pull back** — distance sets power, direction sets angle.
3. **Release** to fire.
4. **Land on the lit pad** to score and keep the run alive.
5. One miss ends the run. `ESC` returns to the menu.

## Settings

- **Cannonball size** (5–15)
- **Number of targets** (5–15)
- **Trajectory guide** — turn it off for a harder, purer aim

Settings and your best score persist between sessions.

## Running

Open the sketch folder in the Processing IDE (4.x) and press Run,
or use the bundled VS Code task (`Run Sketch`).

## Design

The entire look is rendered procedurally — gradient night sky, twinkling starfield,
ringed planet, noise-based mountain silhouettes, and a neon-rimmed launch platform —
in a deep-navy palette with a single luminous cyan accent. Typography is Montserrat.
