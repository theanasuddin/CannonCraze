# Security Policy

Cannon Craze is a small offline game: it requests no permissions, opens no
network connections, and stores only a high score and a settings file on the
local machine. Its attack surface is correspondingly tiny, but reports are
still welcome.

## Supported versions

Only the latest release is supported. Older packages are kept for download but
do not receive fixes.

## Reporting a vulnerability

Please do not open a public issue for anything security-sensitive. Instead use
GitHub's private reporting: **Security tab, then "Report a vulnerability"** on
the repository. Include steps to reproduce and the platform and version you
tested.

You can expect an acknowledgement within a week. If the report is valid, a fix
ships in the next release and you will be credited in the release notes unless
you prefer otherwise.
